# yup-wc

```
NAME:
   wc - print newline, word, and byte counts for each file

USAGE:
   wc [OPTIONS] [FILE...]

      Print newline, word, and byte counts for each FILE, and a total line if
      more than one FILE is specified. A word is a non-zero-length sequence of
      characters delimited by white space.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --lines, -l            print the newline counts (default: false)
   --words, -w            print the word counts (default: false)
   --chars, -m            print the character counts (default: false)
   --bytes, -c            print the byte counts (default: false)
   --max-line-length, -L  print the maximum display width (default: false)
   --help, -h             show help
```
